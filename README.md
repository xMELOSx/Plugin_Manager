# Dionys\_Control

フォルダ内全走査型アプリケーション用汎用プラグインエディッタ試作

# Dionys Control - Function List v0.9.212



Dionys Control は、複数のアプリケーション設定やプラグイン、MOD などの管理を統合し、

仮想的なワークスペース環境を構築するためのデスクトップランチャー兼管理ツールです。



\## 1. コア管理機能 (Core Management)

\- \*\*仮想ワークスペース統合\*\*: 複数のアプリ（例：Minecraft, 各種開発ツール）のストレージパスを単一の UI で管理。

\- \*\*デプロイメントシステム\*\*: シンボリックリンク (Symbolic Link) と物理コピー (Physical Copy) を切り替え可能。

&nbsp; - \*\*シンボリックリンクモード\*\*: ストレージ容量を消費せず、瞬時に環境を切り替え。

&nbsp; - \*\*コピーモード\*\*: 完全な独立環境を構築可能。

\- \*\*自動バックアップ\*\*: ターゲット側のファイルを上書き・削除する前に、タイムスタンプ付きで自動バックアップ。

\- \*\*フォルダ・ファイル構成ルール\*\*:

&nbsp; - \*\*Folder (フォルダごと)\*\*: フォルダをそのまま配置。

&nbsp; - \*\*Flat (直下ファイル)\*\*: フォルダ階層を無視して中身をデプロイ。

&nbsp; - \*\*Tree (再帰的)\*\*: 階層構造を維持したまま特定深度までデプロイ。



\## 2. 組織化・検索機能 (Organization \& Search)

\- \*\*カテゴリー \& パッケージ管理\*\*: アイテムを 2 段階の階層（カテゴリー > パッケージ）で整理。

\- \*\*セグメントベース・クイックタグ\*\*:

&nbsp; - タグをセグメント（グループ）分けして管理。

&nbsp; - 柔軟な AND/OR 検索による高速フィルタリング。

\- \*\*お気に入り (Favorites) システム\*\*:

&nbsp; - スター（☆/★）によるお気に入り管理。

&nbsp; - お気に入りスコア（優先度）による高度なソート機能。

\- \*\*高度な検索バー\*\*: 名前、パス、タグ、メモを横断的に検索。



\## 3. 自動化・効率化ツール (Automation \& Tools)

\- \*\*バッチデプロイメント (Batch Deploy)\*\*: 表示中の全アイテムを 1 クリックで一括デプロイ。

\- \*\*スプリットビュー・デプロイ\*\*: 画面分割を活用し、異なるカテゴリー間のアイテムを直感的にデプロイ。

\- \*\*セキュリティポリシー (Safety Block)\*\*:

&nbsp; - ユーザーが指定した保護パスへの不正な書き込みを自動的にブロック。

\- \*\*実行マネージャー (Executable Manager)\*\*:

&nbsp; - アプリ実行時の引数、起動順序、環境変数を個別に管理。

\- \*\*クイックビュー・マネージャー (Bulk Edit)\*\*:

&nbsp; - アイコン、表示名、タグをグリッド形式で一括編集。

\- \*\*URL リスト機能\*\*: 関連する Web サイト（Wiki, 配布元など）をアイテムごとに管理。



\## 4. ビジュアル \& UX (Visuals \& User Experience)

\- \*\*プレミアム・ダークテーマ\*\*:

&nbsp; - ガラスモーフィズムと深みのあるグレーを基調としたモダンな UI。

\- \*\*フレームレス UI (Frameless Dialog)\*\*:

&nbsp; - Windows のネイティブ枠を排除した、洗練されたカスタムタイトルバー。

\- \*\*カスタマイズ可能なカードサイズ\*\*:

&nbsp; - テキストリスト、ミニ画像、大型画像+テキストの 3 モードをライブ切替。

&nbsp; - スケール、幅、高さ、不透明度を 0.1% 単位で微調整可能。

\- \*\*高 DPI アイコン対応\*\*: 16px から 256px まで、全ての解像度で鮮明に表示。

\- \*\*リッチなマイクロインタラクション\*\*:

&nbsp; - トースト通知、コンテキストメニューのフェード効果、スムーズなレイアウト遷移。



\## 5. その他 (Advanced Options)

\- \*\*多言語対応 (i18n)\*\*: 日本語・英語をネイティブサポート。

\- \*\*設定のエクスポート/インポート\*\*: 環境移行用ツール。

\- \*\*永続化ジオメトリ\*\*: 各ウィンドウのサイズ、位置、検索状態を次回起動時に自動復元。
